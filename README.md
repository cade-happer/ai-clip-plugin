# 🎬 AI 短视频文案插件

浏览任意网页内容，AI 自动提取并一键生成抖音/小红书风格的口播文案。支持多种自然配音下载，自媒体人必备。

## ✨ 功能

- 🔍 **一键提取网页内容** — 浏览文章/新闻，点一下即可提取
- ✍️ **AI 生成口播稿** — 自动改写为抖音/小红书风格短视频文案
- 🎙️ **三种自然配音** — 新闻男声、聊天男声、温柔女声，MP3 下载
- 💰 **免费使用** — 新用户免费额度，Pro 无限使用 ¥19.9/月

## 📥 安装方法

### Edge 浏览器
1. 点击本页 **Code → Download ZIP** 下载并解压
2. 打开 Edge，地址栏输入 `edge://extensions`
3. 打开左侧「**开发人员模式**」
4. 点击「**加载解压缩的扩展**」
5. 选择解压后的文件夹

### Chrome 浏览器
1. 同上 1-2 步，地址栏改为 `chrome://extensions`
2. 打开右上角「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择解压后的文件夹

## 🎙️ 配音说明

配音功能需要本地运行 TTS 服务（Python）：
```bash
pip install edge-tts
python server.py
```
